## Setup
* Docker containers
  * ./setup.sh run
* All initializations
  * ./setup.sh init
* Running
  * node src/api.js
  * See package.json for others scripts  
* Docker containers stop
  * ./setup.sh stop
* Docker containers start
  * ./setup.sh start
